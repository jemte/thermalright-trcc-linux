# TRCC Linux

This is a personal fork of [TRCC Linux by Lexonight1](https://github.com/Lexonight1/thermalright-trcc-linux) that introduces features for my personal use case, a CachyOS system with two USB-connected Thermalright screens (Wonder Vision `87ad:70db` 1600x720 and Trofeo Vision `0416:5302` 1280x480). This fork adds multi-device support and prevents random resets when the screen displays a static background.

Please note that most of the code in my contributions is vibe-coded by Claude. The code is not my own, and is free to use and improve upon. This repository has been built, tested and confirmed to work on my system.
